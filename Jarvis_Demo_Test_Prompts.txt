ARCHITECTURE DEMO TALK TRACK
----------------------------
User Request
    -> Qwen3 (Ollama)
    -> Action Parser
    -> JSON Action
    -> Executor
    -> Tool Registry
    -> Windows Action

Example:
'Open Visual Studio'

Becomes:

{
    "tool": "open_visual_studio",
    "arguments": {}
}

Then Jarvis executes the tool and launches Visual Studio.
